# ImapClient
Imap client - PHP

Config file:
```

// Set SMTP, IMAP and DKIM keys
core/config.php
// Set SMTP, IMAP and DKIM keys

// DKIM key (set if you need or set empty)
core/dkim.txt
// DKIM key

```
